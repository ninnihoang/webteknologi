Questions

  1. What does the server in the client server architecture do?
    1. If my browser is the client in the client server architecture, the server provides the files i see. So the server works as a provider, who transmits and retrieves file to the client.
  2. What does the client do?
    1. The client is the one using the internet, and gain access by using browsers. The browsers can locate and display information from the web. The client requests a web page, then waits for the server to deliver it, and then the browsers renders the package(often an HTML code).
  3. What is an IP address?
    1. An IP adress is an unique adress for each device connected to the internet. It stands for Internet Protocol adress.
  4. What is DNS?
    1. DNS stands for Domain Name Server. A domain name identifies the type of intitution that owns the computer. For example .no is the domain name for norwegain websites. The DNS works as a "phonebook" that map symbolic names (like .no) to their IP adress, so we (humans) dont have to memorize every ip adress when we surf the web.
  5. What is the difference between the Internet and WWW?
    1. The internet is a massive network of networks, a networking infrastructure. It connects millions of computers together globally, forming a network in which any computer can communicate with any other computer as long as they are both connected to the Internet.
    2. The World Wide Web, is a way of accessing information over the medium of the Internet. It is an information-sharing model that is built on top of the Internet. It is a system of interconnected hypertext documents that allowed their users to easily navigate from one topic to another.An entire collection of linked documents is referred to as a web site. The hypertext documents within a web site are known as web pages.
  6. Explain what HTML is
    1. HTML stands for hypertext markup language. It is a language that describes the structure of a page.
